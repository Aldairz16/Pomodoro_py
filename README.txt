==== POMODORO TIMER ====

Esta aplicación te ayuda a gestionar tus sesiones de trabajo y descanso
usando la técnica Pomodoro.

=== CÓMO USAR ===

1. Ejecuta el archivo "ejecutar_pomodoro.bat" haciendo doble clic en él
2. La aplicación se iniciará automáticamente
3. Si es necesario, el script instalará las dependencias requeridas

=== CONFIGURACIÓN ===

- Tiempos: Puedes cambiar la duración de los periodos de trabajo y descanso
- Colores: Personaliza los colores para cada fase
- Sonidos: La aplicación te alertará cuando termine cada fase y 5 segundos antes

=== AÑADIR A LA BARRA DE TAREAS ===

1. Crea un acceso directo a "ejecutar_pomodoro.bat" (clic derecho > Crear acceso directo)
2. Mueve el acceso directo al escritorio o donde prefieras
3. Haz clic derecho en el acceso directo y selecciona "Anclar a la barra de tareas"
